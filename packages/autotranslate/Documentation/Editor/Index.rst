Editor's Guide
==============

How to use:
-----------
1. Open the **Web > Layout** module.
2. Select the page you want to translate.
3. Switch to the "Auto Translate" module from the backend menu.
4. Select the target languages and click **"Start Translation"**.
5. Once finished, the translated content will appear in the page columns.
