=============================================================
Auto Translate
=============================================================

.. extension-key:: autotranslate

:Language: en
:Rendered: |today|

The "Auto Translate" extension allows editors to translate TYPO3 content elements and IRRE records automatically using an external Translation API. It preserves HTML structures, CSS classes, and complex layouts like tables and figures.
